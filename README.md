Чек-лист проверок для сайта
https://opensource-demo.orangehrmlive.com/web/index.php/auth/login :

1. Авторизация

- [ ] Успешная авторизация с валидными кредами (переход на страницу Dashboard)
- [ ] Ошибка авторизации при неправильном пароле
- [ ] Ошибка авторизации при пустых полях "Имя пользователя" и "Пароль"
- [ ] Успешный выход из системы
- [ ] Отображение виджетов на главной странице после авторизации


2. Управление сотрудниками (PIM)

- [ ] Добавление сотрудника
- [ ] Поиск сотрудника по имени/фамилии через фильтр
- [ ] Редактирование личной информации сотрудника (Иконка карандашика)
- [ ] Удаление сотрудника из списка (иконка урны)

3. Заявки на отпуск (Leave)

- [ ] Успешное создание заявки на отпуск
- [ ] Отображение созданной заявки в разделе "Мой отпуск"

4. Администрирование (Admin)

- [ ] Успешное создание пользователя
- [ ] Отображение созданного пользователя в списке
- [ ] Удаление пользователя

5. Подбор персонала (Recruitment)

- [ ] Поиск нужного сотрудника
- [ ] Просмотр информации по сотруднику (Иконка глазик)

6. Моя информация (My info)

- [ ] Переход к просмотру информации о себе
